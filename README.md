# ansible-go-ubuntu

# Description

**ansible-go-ubuntu** is a simple [Ansible](https://www.ansible.com/) role for installing [Go](https://golang.org/) for development on an Ubuntu machine.


# Usage

Within Ansible playbook:

```
...
roles:
  - ansible-go-ubuntu
...
```
